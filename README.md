# Rock Paper Scissors Game

A simple Rock Paper Scissors game built using HTML, CSS, and JavaScript. The game allows users to play against the computer, keeps track of scores, displays the result of each round, and includes a reset button to start over at any time.

## Features

Play against the computer with randomly generated choices.

Track both player and computer scores in real time.

View the result of each round instantly.

Reset the game whenever you want.

Responsive design that works on desktop and mobile devices.

## Technologies Used

HTML5, CSS3, and JavaScript.

## How to Play

Select Rock, Paper, or Scissors. The computer will automatically generate a random choice. The winner is determined according to the standard Rock Paper Scissors rules, and the scores are updated immediately. Use the reset button to restart the game.
